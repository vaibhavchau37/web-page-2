1. Typography Showcase

Serif Font

Elegant and traditional for print-like feel

font-family: Georgia, serif;

Sans-Serif Font

Clean and modern for digital interfaces

font-family: 'Poppins', sans-serif;

Monospace Font

Technical and precise for code readability

font-family: 'Courier New', monospace;

Cursive Font

Creative and artistic for decorative text

font-family: 'Brush Script MT', cursive;

Fantasy Font

Bold and decorative for impactful titles

font-family: Impact, fantasy;

Combined Styles

Mix of bold, italic & underline effects

font-weight: bold;
font-style: italic;
text-decoration: underline;

2. Background Repeat Control

Repeat (default)

Tiles in both directions

background-repeat: repeat;

Repeat-X

Horizontal stripes only

background-repeat: repeat-x;

Repeat-Y

Vertical stripes only

background-repeat: repeat-y;

No-Repeat

Single centered image

background-repeat: no-repeat;
background-position: center;

3. Interactive Link States

Normal Link

Default appearance before interaction

a:link { ... }
Hover Effect

When mouse pointer is over link

a:hover { ... }
Active State

While link is being clicked

a:active { ... }
Visited Link

After link has been clicked

a:visited { ... }

4. Element Visibility

Visible

visibility: visible;

Hidden

visibility: hidden;

Display None

display: none;

Key Differences:

  • visibility: hidden - Element is hidden but still occupies space in layout
  • display: none - Element is completely removed from document flow
